## 🔧 Hibaelhárítás

### ❌ "Nem találom a szervert"

**Megoldások:**
1. Ellenőrizd az IP címet: `31.214.216.164:5040`
2. Add hozzá a Favorites-hez
3. Frissítsd a szerver listát (F5)
4. Ellenőrizd, hogy a szerver online-e

### ❌ "Connection timeout"

**Megoldások:**
1. Ellenőrizd az internet kapcsolatot
2. Tűzfal beállítások (5040 UDP engedélyezése)
3. Próbálj meg újra csatlakozni
4. VPN kikapcsolása (ha van)

### ❌ "Wrong password"

**Megoldások:**
1. Ellenőrizd, hogy nincs-e szerver jelszó
2. Ha van, kérd el a Discord-on
3. Admin jelszó ≠ szerver jelszó

### ❌ "Version mismatch"

**Megoldások:**
1. Frissítsd az ARK: Survival Ascended-et
2. Steam → ARK → Properties → Updates
3. Verify Integrity of Game Files
4. Indítsd újra a játékot

### ❌ "Modok hiányoznak"

**Megoldások:**
1. Iratkozz fel a szerver modjaihoz:
   - Mod Manager → Browse → Keress a mod nevére
2. Várj, amíg az összes mod letöltődik
3. Indítsd újra az ARK-ot
4. Próbálj újra csatlakozni
